## Examples:

"I hate the product"

**Number of Words:** 4

**Total Score:** -37

**Average Word Score:** -9.25

```diff
- This sentence seems more Negative
```


___


"I love the product"

**Number of Words:** 4

**Total Score:** 86

**Average Word Score:** 21.5

```diff
+ This sentence seems more Positive
```
